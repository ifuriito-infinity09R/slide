### Code Presenting
コードブロックのハイライトができます。
```js
console.log("@[1]と書くと、1行目がハイライトされます。");
console.log("@[2-4]といったように");
console.log("複数行選択も");
console.log("可能です。");
console.log("eof");
```
@[1](ハイライト行の解説も記述できます。)
@[2-4](コメントは[]の右に記述します。)
@[4](お試し）
